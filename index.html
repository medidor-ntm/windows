Script de Reinstalação

Medidor Educação Conectada (Windows)

Baixar Script (medidorwindows.bat)

Instruções de Execução

  1. Faça o download do arquivo clicando no botão azul acima.
  2. Abra sua pasta de "Downloads" e localize o arquivo medidorwindows.bat.
  3. Clique com o botão direito do mouse sobre o arquivo.
  4. No menu que aparecer, selecione a opção "Executar como administrador".
  6. Se uma janela de permissão (UAC) aparecer, clique em "Sim" para permitir.
  7. O script irá rodar automaticamente para limpar a instalação antiga e baixar o novo instalador.

A permissão de administrador é necessária para que o script possa remover arquivos antigos do sistema e concluir a reinstalação corretamente.
